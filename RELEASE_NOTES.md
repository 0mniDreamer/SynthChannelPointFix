# SynthChannelPoints v1.0.2 — release post draft

**Twitch channel point redemptions are back!**

Twitch removed the service Synth Riders used for channel point rewards back in April 2025 — chat commands kept working, but point redemptions (song requests via points, effect rewards) silently died. SynthChannelPoints bridges Twitch's new event system into the game using your existing in-game Twitch login. No setup, no tokens, no new accounts: install the DLL, link Twitch in-game like you always did, and name your channel point rewards after the chat commands (e.g. a reward titled `!srr` with viewer text input for song requests).

Requires MelonLoader 0.7+ and Twitch Affiliate/Partner (channel points are a Twitch feature). All the game's original behavior — queue limits, cooldowns, bot replies — works exactly as before.
